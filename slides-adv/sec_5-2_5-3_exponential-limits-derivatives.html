    Subsection Derivative of Exponential

    For each new function, we need to use the definition of the derivative to discover the corresponding derivative rule.
    \begin{equation*} f'(x) = \lim_{h \to 0}\frac{f(x+h)-f(x)}{h} \end{equation*}
    When \(f(x) = b^x\text{,}\) we use the exponent sum rule to rewrite \(f(x+h) = b^{x+h} = b^x \cdot b^h\text{:}\)
    \begin{align*} f'(x) &= \lim_{h \to 0} \frac{b^{x+h}-b^x}{h} = \lim_{h \to 0} \frac{b^x b^h - b^x}{h}\\ &= \lim_{h \to 0} \frac{b^x(b^h-1)}{h} = b^x \cdot \lim_{h \to 0} \frac{b^h - 1}{h} \end{align*}
    Consequently, the derivative of an elementary exponential is always some constant \(L(b)\) times the original function \(b^x\text{,}\) where
    \begin{equation*} L(b) = \lim_{h \to 0}\frac{b^h-1}{h} \end{equation*}

    Subsection Natural Base \(e\)

    The value of \(L(b)\) is exactly the value of the derivative of \(b^x\) at \(x=0\text{.}\) Different bases have different slopes at this point.
    ðŸ”—
    The base \(e\) is that number such that \(L(b) = 1\) so that the tangent line is exactly \(y=1+x\text{:}\)
    \begin{equation*} \lim_{h \to 0} \frac{e^h - 1}{h} = \lim_{x \to 0} \frac{e^x - 1}{x} = 1. \end{equation*}
    The value as a decimal approximation is \(e \approx 2.71828183\text{.}\)
    ðŸ”—
    Since \(L(e) = 1\text{,}\) we have the remarkable derivative rule \(\frac{d}{dx}[e^x] = e^x\text{.}\) Incorporating the chain rule, for any formula \(u\text{,}\) we have
    \begin{equation*} \frac{d}{dx}[e^u] = e^u \cdot u'\text{.} \end{equation*}

    Subsection The Natural Logarithm

    Now that we have defined the natural base \(e\text{,}\) we can also define the natural logarithm, which is the inverse function to \(y=e^x\text{:}\)
    \begin{equation*} \ln(e^x) = x \qquad \text{and} \qquad e^{\ln(x)} = x\text{.} \end{equation*}

    Subsection The Natural Logarithmâ€™s Derivative

    To find \(\ln'(x)\text{,}\) we will do the same approach as for all inverse functionsâ€”use implicit differentiation:
    \begin{gather*} y = \ln(x) \quad \Leftrightarrow \quad e^y = x\\ \frac{d}{dx}[e^y] = \frac{d}{dx}[x]\\ e^y \cdot y' = 1\\ y' = \frac{1}{e^y} = \frac{1}{x} \end{gather*}
    That is, \(\ln'(x) = \frac{1}{x}\text{,}\) and for any expression \(u\text{,}\) the chain rule gives:
    \begin{equation*} \frac{d}{dx}\ln(u) = \ln'(u) \cdot u' = \frac{1}{u} \cdot u' = \frac{u'}{u} \end{equation*}
