    Subsection Exponential Models

    A more general exponential function has a constant multiple or amplitude factor,
    \begin{equation*} f(x) = A \cdot b^x\text{.} \end{equation*}
    We will later learn about the natural base \(e\) and discover that every exponential function can alternatively be written
    \begin{equation*} f(x) = A e^{kx}\text{,} \end{equation*}
    where the value of \(k\) and the base \(b\) are related by \(k = \ln(b)\text{.}\)
    ðŸ”—
    • When the base \(b\) is greater than 1 (or \(k > 0\)), we say we have exponential growth.

      ðŸ”—

    Subsection Logarithms as Inverses

    Recall that an inverse function allows us to solve for the \(x\) that makes \(f(x) = c\) true,
    \begin{equation*} x = f^{-1}(c)\text{.} \end{equation*}
    When we want to solve for an exponent, \(b^x = c\text{,}\) we want the inverse of the elementary exponential function.
    ðŸ”—
    For every positive base \(b > 0\) with \(b \ne 1\) and \(c > 0\text{,}\) the equation
    \begin{equation*} b^x = c \end{equation*}
    has a unique solution. (That is, the elementary exponential functions are one-to-one.) The inverse function finds this solution and is called the logarithm with base \(b\text{,}\)
    \begin{equation*} x = \log_b(c)\text{.} \end{equation*}
    ðŸ”—
    Because \(b^x > 0\) for all \(x\text{,}\) the domain of \(\log_b(x)\) requires \(x > 0\text{.}\) (The logarithm is undefined for zero and negative values.)
    ðŸ”—
    ðŸ”—
